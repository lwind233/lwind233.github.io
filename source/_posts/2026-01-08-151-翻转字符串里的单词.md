用双指针记录一下每个单词的头尾，倒序遍历字符串将所有的单词挨个提取出来即可

java没法修改字符串所以没法做原地算法

```java
class Solution {
    public String reverseWords(String s) {
        int len = s.length();
        int st = len-1;
        int ed = len-1;
        StringBuilder result = new StringBuilder();
        while(st>=0){
            if(s.charAt(ed)==' '){
                while(ed>=0 && s.charAt(ed)==' ') ed--;
                st = ed;
                if(st==-1 && s.charAt(0)==' ') break;
            }
            while(st>=0 && s.charAt(st)!=' ') st--;
            result.append(s.substring(st+1,ed+1));
            result.append(' ');
            ed = st;
        }
        String tmpResult = result.toString();
        return tmpResult.substring(0,tmpResult.length()-1);
    }
}
```
