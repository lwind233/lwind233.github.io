题目要求的是平衡二叉搜索树，也就是两边的节点数一致（或者相差1）。

也就是对这个数组，根节点肯定是中点，左子树是左半部分，右子树是右半部分。

按这个思路递归解决即可，每次以当前子数组的中间值为根节点，然后递归构建左右子树，返回根节点。

```java
class Solution {
    public TreeNode sortedArrayToBST(int[] nums) {
        return traversal(nums,0,nums.length-1);
    }

    public TreeNode traversal(int[] nums, int s, int e){
        if(s>e) return null;
        int mid = (s+e)/2;
        TreeNode root = new TreeNode(nums[mid]);
        root.left = traversal(nums,s,mid-1);
        root.right = traversal(nums,mid+1,e);
        return root;
    }
}
```
