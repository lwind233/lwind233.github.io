先深度优先搜索遍历整颗树，每次遍历都是一条路径，在这条路径上记录前缀和，查找前缀和频率统计表中是否存在前缀和使二者之差等于目标值（即存在子数组和等于目标值），随后将前缀和记录存入前缀和频率统计表中，对左右子节点进行搜索，搜索完毕后从前缀和频率统计表中删除掉当前的前缀和统计信息，实现回溯。

```java
class Solution {
    int result=0;
    Map<Long,Long> bsumPath;

    public int pathSum(TreeNode root, int targetSum) {
        if(root==null) return 0;
        bsumPath = new LinkedHashMap<>();
        bsumPath.put(0L,1L);
        backTrace(root,targetSum,0L);
        return result;
    }


    public void backTrace(TreeNode root, int targetSum, long bsum){
        if(root==null) return;
        bsum += root.val;
        result+=bsumPath.getOrDefault(bsum-targetSum,0L);
        bsumPath.put(bsum,bsumPath.getOrDefault(bsum,0L)+1);
        backTrace(root.left,targetSum,bsum);
        backTrace(root.right,targetSum,bsum);
        bsumPath.put(bsum,bsumPath.get(bsum)-1);
    }
}
```
