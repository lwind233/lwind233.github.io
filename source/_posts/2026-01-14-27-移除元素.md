遍历数组，记录offset，offset代表当前位置跳过了多少个等于val的值

```java
class Solution {
    public int removeElement(int[] nums, int val) {
        int offset=0;
        int len=nums.length;
        for(int i=0;i<=len-1;i++){
            while(i+offset<=len-1 && nums[i+offset]==val){
                offset++;
            }
            if(offset+i>len-1) break;
            nums[i] = nums[i+offset];
        }
        return len-offset;
    }
}
```
