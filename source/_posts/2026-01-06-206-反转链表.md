## 递归解法

每层传入当前节点和当前节点的父节点

如果当前节点的下一个节点是null，也就是尾节点

那当前节点就是新的头结点

然后对每一层，让当前节点指向父节点

最后将原始的头结点指向null

```java
class Solution {
    ListNode fin=null;


    public ListNode reverseList(ListNode head) {
        if(head==null || head.next==null) return head;
        traversal(head,head.next);
        head.next=null;
        return fin;
    }


    public void traversal(ListNode pre, ListNode cur){
        if(cur.next!=null){
            traversal(cur,cur.next);
            cur.next=pre;
        }else{
            fin=cur;
            cur.next=pre;
        }
    }
}
```

## 迭代解法

还是经典维护双指针，或者说三指针，记录好当前节点的下一个节点，然后让当前节点指向父节点，最后更新父节点指针和当前节点指针的位置

```java
class Solution {
    public ListNode reverseList(ListNode head) {
        ListNode pre = null;
        ListNode cur = head;
        ListNode tmp;
        while(cur!=null){
            tmp = cur.next;
            cur.next = pre;
            pre = cur;
            cur = tmp;
        }
        return pre;

    }
}
```
