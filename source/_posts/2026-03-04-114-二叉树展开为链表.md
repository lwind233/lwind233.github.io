要求O(1)空间复杂度，就不能用栈

迭代法前序遍历一遍树，保留一个前导节点prev，每次让prev指向当前节点cur，将cur的两个子节点入栈，然后更新prev，并断掉cur的左节点。

```java
class Solution {
    public void flatten(TreeNode root) {
        Stack<TreeNode> stack = new Stack<>();
        TreeNode pre = new TreeNode();
        if(root==null) return;
        stack.push(root);

        while(!stack.isEmpty()){
            TreeNode tmp = stack.pop();
            pre.right = tmp;
            if(tmp.right!=null) stack.push(tmp.right);
            if(tmp.left!=null) stack.push(tmp.left);
            pre = tmp;
            tmp.left=null;
        }
    
    }
}

```
