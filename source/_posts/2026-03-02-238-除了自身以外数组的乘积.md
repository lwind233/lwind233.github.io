先计算第i位元素的前缀乘积（从第一位元素到第i位元素的乘积），然后计算第i位元素的后缀乘积（从第i位元素到最后一位元素的乘积）

随后对第i位元素，除了其的乘积为第i-1位元素的前缀乘积与第i+1位元素的后缀乘积。左边界的前缀乘积为1，右边界的后缀乘积为1。

```java
class Solution {
    public int[] productExceptSelf(int[] nums) {
        int[] pre = new int[nums.length];
        int[] post = new int[nums.length];
        int[] result = new int[nums.length];
        for(int i=0;i<=nums.length-1;i++){
            if(i==0) pre[i] = nums[i];
            else pre[i] = nums[i]*pre[i-1];
        }

        for(int i=nums.length-1;i>=0;i--){
            if(i==nums.length-1) post[i] = nums[i];
            else post[i] = nums[i]*post[i+1];
        }

        for(int i=0;i<=nums.length-1;i++){
            if(i==0){
                result[i] = post[i+1];
            }else if(i==nums.length-1){
                result[i] = pre[i-1];
            }else{
                result[i] = pre[i-1]*post[i+1];
            }
        }
        
        return result;
    }
}
```
