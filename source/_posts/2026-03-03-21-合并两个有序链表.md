想麻烦了，直接套了合并k个有序链表的代码，实际上不用这么麻烦，就两个链表的话直接比大小就好不用塞到堆里

```java
class Solution {
    public ListNode mergeTwoLists(ListNode list1, ListNode list2) {
        PriorityQueue<ListNode> pQ = new PriorityQueue<>(2,(a,b)->a.val-b.val);
        if(list1==null) return list2;
        if(list2==null) return list1;
        ListNode result = new ListNode();
        ListNode cur = result;

        pQ.offer(list1);
        pQ.offer(list2);
        while(!pQ.isEmpty()){
            ListNode tmp = pQ.poll();
            if(tmp.next!=null) pQ.offer(tmp.next);
            cur.next = tmp;
            cur = tmp;
        }

        return result.next;
    }
}
```
