拿一个指针当探头

两个指针间隔n个距离

当探头指到链表尾部时，前面的指针刚好是待删除节点的父节点

```java
class Solution {
    public ListNode removeNthFromEnd(ListNode head, int n) {
        ListNode tHead = new ListNode();
        tHead.next = head;
        int cnt = 0;
        ListNode pre = tHead;
        ListNode cur = head;
        while(cnt<n){
            cur = cur.next;
            cnt++;
        }


        while(cur!=null){
            cur = cur.next;
            pre = pre.next;
        }
        pre.next = pre.next==null ? null : pre.next.next;


        return tHead.next;


    }
}
```
