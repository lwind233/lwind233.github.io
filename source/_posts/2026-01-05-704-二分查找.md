二分感觉没啥好说的，单调数列，一个左指针一个右指针，中间值直接取二者平均值，然后遍历直到找到目标即可
（如果数列长度过大考虑l+r-l/2防止溢出）

```java
class Solution {
    public int search(int[] nums, int target) {
        int right = nums.length-1;
        int left = 0;
        int mid = (right+left)/2;
        while(left<=right){
            if(nums[mid]>target){
                right = mid-1;
                mid = (right+left)/2;
            }else if(nums[mid]<target){
                left = mid+1;
                mid = (right+left)/2;
            }else if(nums[mid]==target){
                return mid;
            }
        }
        return -1;
    }
}
```
