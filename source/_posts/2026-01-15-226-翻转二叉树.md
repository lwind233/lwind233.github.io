递归做，先递归处理左右子树

左右子树处理完后，将当前节点的左右子树交换即可

此时左右子树里的左右子树已经交换，当前节点的左右子树也交换，递归完全

```java
class Solution {
    public TreeNode invertTree(TreeNode root) {
        traversal(root);
        return root;
    }

    public void traversal(TreeNode root){
        if(root==null) return;
        traversal(root.left);
        traversal(root.right);
        TreeNode tmp = root.left;
        root.left = root.right;
        root.right = tmp;
    }
}
```
