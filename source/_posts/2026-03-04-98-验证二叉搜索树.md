## 递归法

二叉搜索树的条件：左节点小于根节点小于右节点，同时根节点有一个根据父节点限定的取值范围

每一层先判断根节点是否符合范围，然后判断左右子树是不是二叉搜索树，再判断左节点小于根节点小于右节点

给子树限定范围时，右子树的最小值是根节点的值，左子树的最大值是根节点的值
```java
class Solution {
    public boolean isValidBST(TreeNode root) {
        return fixIsValidBST(root,Long.MAX_VALUE,Long.MIN_VALUE);
    }

    public boolean fixIsValidBST(TreeNode root, long max, long min){
        boolean left=true;
        boolean right=true;
        if((long) root.val<=min || (long) root.val>=max) return false;

        if(root.left!=null){
            left = root.val>root.left.val && fixIsValidBST(root.left,Math.min(root.val,max),min);
        }
        if(root.right!=null){
            right = root.val<root.right.val && fixIsValidBST(root.right,max,Math.max(root.val,min));
        }

        return left&&right;
    }
}
```
