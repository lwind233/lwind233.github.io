为两个字符串建立哈希表，比较哈希表是否一致即可

节省内存的话可以只建立一个哈希表，第一次建立时统计字符频数，第二次减去字符频数，然后看哈希表内是不是每个键都是0

注意Integer比较时要用equals而不能用==

```java
class Solution {
    public boolean isAnagram(String s, String t) {
        HashMap<Character,Integer> hashMapS = new HashMap<>();
        HashMap<Character,Integer> hashMapT = new HashMap<>();


        int sLen = s.length();
        int tLen = t.length();


        if(sLen!=tLen) return false;


        for(int i=0;i<=sLen-1;i++){
            hashMapS.put(s.charAt(i),hashMapS.getOrDefault(s.charAt(i),0)+1);
            hashMapT.put(t.charAt(i),hashMapT.getOrDefault(t.charAt(i),0)+1);
        }


        for(Character c : hashMapS.keySet()){
            if(!hashMapS.get(c).equals(hashMapT.getOrDefault(c,-1))) return false;
        }


        return true;
    }
}
```
