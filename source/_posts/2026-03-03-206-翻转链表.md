## 递归法

递归函数处理after后面的结点，使其变成从尾部指向头部

然后先递归后面的部分，当前步骤只需要将尾节点指向头结点即可

```java
class Solution {
    ListNode nHead=null;
    public ListNode reverseList(ListNode head) {
        traversal(null,head);
        return nHead;
    }

    public void traversal(ListNode pre, ListNode after){
        if(after==null){
            nHead=pre;
            return;
        }
        traversal(after,after.next);
        after.next=pre;
    }
}
```

## 三指针法

维护三个指针，pre记录当前的头结点，cur记录当前的尾节点，tmp记录尾结点的下一个结点。
先记录好下一个结点，然后利用pre和cur翻转两个结点之间的指向关系，最后更新cur和pre的位置。
当cur指向null时，pre指向的就是新的头结点


```java
class Solution {
    public ListNode reverseList(ListNode head) {
        ListNode pre = null;
        ListNode cur = head;
        ListNode tmp;
        while(cur!=null){
            tmp = cur.next;
            cur.next = pre;
            pre = cur;
            cur = tmp;
        }
        return pre;
        
    }
}
```
