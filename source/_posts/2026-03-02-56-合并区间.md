先把区间按左端点进行排序，然后遍历区间，判断新的区间的左端点是否在旧区间之中，是的话合并两个区间，不是的话将老区间加入结果列表，根据新区间进行判断。

区间排序可以选择直接对数组进行排序，利用Array.sort，或者采用优先队列将下标存入堆中。

```java
class Solution {
    public int[][] merge(int[][] intervals) {
        PriorityQueue<Integer> pQ = new PriorityQueue<>(intervals.length, (a,b)->intervals[a][0]-intervals[b][0]);
        for(int i=0;i<=intervals.length-1;i++){
            pQ.offer(i);
        }

        List<int[]> ls = new ArrayList<>();
        int st,ed;
        int tmpSt,tmpEd;
        int idx =  pQ.poll();
        st = intervals[idx][0];
        ed = intervals[idx][1];
        while(!pQ.isEmpty()){
            idx = pQ.poll();
            tmpSt = intervals[idx][0];
            tmpEd = intervals[idx][1];
            if(tmpSt<=ed){
                st = Math.min(tmpSt,st);
                ed = Math.max(tmpEd,ed);
            }else{
                ls.add(new int[]{st,ed});
                st = tmpSt;
                ed = tmpEd;
            }
        }
        ls.add(new int[]{st,ed});

        int[][] result = ls.toArray(new int[ls.size()][]);

        return result;
    }
}
```
