别想麻烦了，每一步就俩条件

如果左括号数量比n小，说明当前情况可以放一个左括号

如果右括号数量比左括号小，说明当前情况可以放一个右括号

只要左括号等于右括号等于n，那肯定就是一个合法的情况，同时每个位置能否放左右括号都被遍历过，不重不漏

```java
class Solution {
    StringBuilder path = new StringBuilder();
    List<String> result = new ArrayList<>();
    public List<String> generateParenthesis(int n) {
        traceBack(n,0,0);
        return result;
    }
    void traceBack(int n, int left, int right){
        if(right==n && left==right){
            result.add(path.toString());
            return;
        }
        if(left<n){
            path.append('(');
            traceBack(n,left+1,right);
            path.deleteCharAt(path.length()-1);
        }
        if(right<left){
            path.append(')');
            traceBack(n,left,right+1);
            path.deleteCharAt(path.length()-1);
        }

    }
}
```
