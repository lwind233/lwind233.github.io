把k个链表按头结点值排序塞到一个最小堆里，然后每次取出一个链表头结点，把剩下的部分塞回到堆里去。每次获取结点的时间复杂度是logk，要取n次也就是nlogk

```java
class Solution {
    public ListNode mergeKLists(ListNode[] lists) {
        PriorityQueue<ListNode> pQ = new PriorityQueue<>(1,(a,b)->a.val-b.val);
        int k = lists.length;
        if(k==0) return null;

        for(int i=0;i<=k-1;i++){
            if(lists[i]!=null) pQ.offer(lists[i]);
        }

        ListNode result = new ListNode(0);
        ListNode cur=result;
        while(!pQ.isEmpty()){
            ListNode tmp = pQ.poll();
            if(tmp.next!=null) pQ.offer(tmp.next);
            cur.next = tmp;
            cur = tmp;
        }

        return result.next;
    }
}
```
