用哈希表记录过去出现过的数字，如果该数字出现过说明进入了循环，也就不可能是快乐数。否则自然会变成1

```java
class Solution {
    public boolean isHappy(int n) {
        HashMap<Integer, Integer> path = new HashMap<>();
        while(n!=1){
            if(path.containsKey(n)) return false;
            path.put(n,1);
            int tmp = n;
            n = 0;
            int tn;
            while(tmp!=0){
                tn = tmp%10;
                tmp = tmp/10;
                n+=tn*tn;
            }
        }
        return true;
    }
}
```
