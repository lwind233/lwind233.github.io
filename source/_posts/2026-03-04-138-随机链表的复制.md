## 节点交替法

最优解法

在链表中的每一个结点后穿插一个与之值一样的新节点，那么新结点的random就是原结点random的下一个值，这样就可以给新结点的random赋好值，随后再把新结点串在一起即可。

```java
class Solution {
    public Node copyRandomList(Node head) {
        Node p,cur;
        cur = head;
        while(cur!=null){
            p = new Node(cur.val);
            p.next = cur.next;
            cur.next = p;
            cur = p.next;
        }
        
        cur = head;
        while(cur!=null){
            if(cur.random!=null){
                cur.next.random = cur.random.next;
            }
            cur = cur.next.next;
        }

        Node result = new Node(0);
        Node pre = result; 
        cur = head;
        while(cur!=null){
            pre.next = cur.next;
            pre = cur.next;
            cur.next = pre.next;
            cur = cur.next;
        }

        return result.next;
    }
}
```

## 哈希表法

建立两个哈希表，一个是原链表的node:idx哈希表，另一个是新链表的idx:node哈希表。仿照原链表构建完新链表的同时记录好原链表每个node对应的idx，然后再遍历一遍原链表找出random对应的下标，为新链表赋值


```java
class Solution {
    public Node copyRandomList(Node head) {
        Map<Node, Integer> map1 = new LinkedHashMap<>();
        Map<Integer, Node> map2 = new LinkedHashMap<>();

        Node thead = new Node(0);
        Node cur = head;
        Node cur2 = thead;

        int count=0;
        while(cur!=null){
            count++;
            Node tmp = new Node(0);
            tmp.val = cur.val;

            map1.put(cur,count);
            map2.put(count,tmp);

            cur2.next = tmp;
            cur2 = tmp;
            cur = cur.next;
        }

        cur = head;
        cur2 = thead.next;
        while(cur!=null){
            if(cur.random!=null){
                int idx = map1.get(cur.random);
                cur2.random = map2.get(idx);
            }

            cur = cur.next;
            cur2 = cur2.next;
        }

        return thead.next;
    }
}
```
