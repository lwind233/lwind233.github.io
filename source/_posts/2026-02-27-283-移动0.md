维护快慢指针，快指针只指向当前不为0的值，慢指针只指向当前需要赋值的值。最后的快慢指针之间的偏移量就是0的数量

```java
class Solution {
    public void moveZeroes(int[] nums) {
        int offset = 0;
        int l = nums.length;
        for(int i=0;i<=l-1;i++){
            while(i+offset<=l-1 && nums[i+offset]==0) offset++;
            if(i+offset>l-1) break;
            nums[i] = nums[i+offset];
        }

        for(int i=0;i<=offset-1;i++){
            nums[l-1-i]=0;
        }
    }
}
```
