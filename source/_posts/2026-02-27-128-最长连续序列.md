先去重，这里面重复元素只会影响解题

然后只从起点开始找，找结尾最长的序列即可

```java
class Solution {
    public int longestConsecutive(int[] nums) {
        Set<Integer> set = new HashSet<>();
        for (int x : nums) set.add(x);

        int best = 0;
        for (int x : set) {
            if (!set.contains(x - 1)) {      // 只从起点开始扩展
                int y = x;
                while (set.contains(y)) y++;
                best = Math.max(best, y - x);
            }
        }
        return best;
    }
}
```
