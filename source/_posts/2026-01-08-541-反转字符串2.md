我认为这道题没啥记得必要，但做都做了

用rev记录当前状态，每k个字符后判断是否要反转，要反转就从尾到头添加字符，否则从头到尾添加字符，最后处理一下尾部。

```java
class Solution {
    public String reverseStr(String s, int k) {
        StringBuilder result = new StringBuilder();
        StringBuilder tmp;
        boolean rev = true;
        int count=0;
        for(int i=0;i<=s.length()-1;i++){
            count++;
            if(count==k && rev){
                for(int j=0;j<=k-1;j++){
                    result.append(s.charAt(i-j));
                }
                rev = false;
                count=0;
            }else if(count==k && !rev){
                for(int j=k-1;j>=0;j--){
                    result.append(s.charAt(i-j));
                }
                rev = true;
                count=0;
            }
        }


        if(rev) for(int j=0;j<=count-1;j++) result.append(s.charAt(s.length()-1-j));
        else for(int j=count-1;j>=0;j--) result.append(s.charAt(s.length()-1-j));


        return result.toString();
    }
}
```
