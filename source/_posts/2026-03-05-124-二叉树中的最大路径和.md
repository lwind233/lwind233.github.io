遍历左右子树，返回子树延伸到当前结点的最大路径和，同时维护一个全局最大路径和。

每一步，比较左右子结点继续延伸到当前根节点的路径和以及当前根节点的值，判断是从当前结点开始的新路径路径和最大，还是沿着左右子树往上延伸的路径和更大，随后比较以根节点位拐点的路径和会不会比全局最大路径和大。

同时借助Math.max(,0)，如果左右子树的路径和贡献为负数直接不考虑。

```java
class Solution {
    private int ans = Integer.MIN_VALUE;

    public int maxPathSum(TreeNode root) {
        dfs(root);
        return ans;
    }

    // 返回：从当前节点出发，向父节点“贡献”的最大路径和（只能选一边 or 不选）
    private int dfs(TreeNode node) {
        if (node == null) return 0;

        int leftGain  = Math.max(0, dfs(node.left));   // 负的就不选
        int rightGain = Math.max(0, dfs(node.right));  // 负的就不选

        // 以 node 为“拐点”的最大路径：左 + node + 右（更新全局答案）
        int through = node.val + leftGain + rightGain;
        ans = Math.max(ans, through);

        // 返回给父亲：只能走一边（node + max(left,right)）
        return node.val + Math.max(leftGain, rightGain);
    }
}
```
