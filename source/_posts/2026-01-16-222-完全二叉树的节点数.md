直接遍历一遍树就行了

```java
class Solution {
    int count;
    public int countNodes(TreeNode root) {
        traversal(root);
        return count;
    }

    void traversal(TreeNode root){
        if(root==null) return;
        count++;
        traversal(root.left);
        traversal(root.right);
    }
}
```
