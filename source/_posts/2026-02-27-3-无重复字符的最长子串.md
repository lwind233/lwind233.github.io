窗口是滑动的，右指针扩展，遇到问题后收缩窗口，左指针右移直到窗口内元素满足条件，随后继续拓展窗口

```java
class Solution {
    public int lengthOfLongestSubstring(String s) {
        int l=0;
        if(s.length()==0) return 0;

        int[] hmp = new int[128];
        int left=0;
        int right=0;
        hmp[s.charAt(0)]=1;
        int maxL = 1;
        right++;

        for(;right<=s.length()-1;right++){
            hmp[s.charAt(right)] = hmp[s.charAt(right)]+1;
            while(hmp[s.charAt(right)]>1){
                hmp[s.charAt(left)] = hmp[s.charAt(left)]-1;
                left++;
            }
            maxL = right-left+1>maxL?right-left+1:maxL;
        }

        return maxL;
    }
}
```
