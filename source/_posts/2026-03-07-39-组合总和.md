每层回溯做两个选择：

* 将当前数字添加到路径里
* 当前数字不再使用，接下来的遍历从下一个数字开始

当数字下标超出候选范围，或者总和大于目标值时回退

当数字总和等于目标值时记录路径

```java
class Solution {
    LinkedList<Integer> path;
    int sum;
    List<List<Integer>> result;
    public List<List<Integer>> combinationSum(int[] candidates, int target) {
        path = new LinkedList<>();
        result = new ArrayList<>();
        sum = 0;
        traceBack(candidates,target,0);
        return result;
    }

    void traceBack(int[] candidates, int target, int idx){
        if(idx==candidates.length) return;
        if(sum>target) return;
        else if(sum==target){
            result.add(new ArrayList<>(path));
            return;
        }else{
            traceBack(candidates,target,idx+1);

            sum+=candidates[idx];
            path.add(candidates[idx]);
            traceBack(candidates,target,idx);
            path.removeLast();
            sum-=candidates[idx];
        }
    }
}
```
