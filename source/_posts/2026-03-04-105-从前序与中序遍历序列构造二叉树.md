先序遍历可以告诉我们第一个遍历的头结点是什么

中序遍历序列中，头结点左侧的子数组就是左子树的中序遍历序列，右侧的子数组就是右子树的中序遍历序列

那么只需要每次找出中序遍历序列中，在先序遍历里序号最小的节点作为头节点，并划分出左右子树，最后递归构建二叉树即可

> 但是，这个实现并不是最优的。最优的实现是根据先序遍历序列找出头结点（也就是开头的元素），然后为中序序列建立哈希表，找出其下标，随后根据中序序列得出左右子树的节点数，对先序序列和中序序列进行划分，交给递归函数进行子树建立，这样的好处是时间复杂度更低，问题是这个节点数的计算可能有问题（我第一次就在这里踩了坑）

```java
class Solution {
    public TreeNode buildTree(int[] preorder, int[] inorder) {
        Map<Integer,Integer> pMap = new LinkedHashMap<>();
        int length = preorder.length;
        for(int i=0;i<=length-1;i++){
            pMap.put(preorder[i],i);
        }
        return traversal(pMap,inorder,0,inorder.length-1);
    }

    public TreeNode traversal(Map<Integer,Integer> pMap, int[] inorder, int s, int e){
        if(s>e) return null;

        int minIdx = Integer.MAX_VALUE;
        int idx = 0;
        for(int i=s;i<=e;i++){
            int tmp = pMap.get(inorder[i]);
            if(tmp<minIdx){
                minIdx = tmp;
                idx = i;
            }
        }

        TreeNode root = new TreeNode(inorder[idx]);
        root.left = traversal(pMap,inorder,s,idx-1);
        root.right = traversal(pMap,inorder,idx+1,e);
        return root;
    }
}
```
