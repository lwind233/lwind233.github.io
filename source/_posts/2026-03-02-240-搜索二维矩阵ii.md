从右上角看可以将矩阵转换为类似二叉搜索树的形式，对当前元素而言所有小于其的元素都在左边，大于其的元素都在下面

```java
class Solution {
    public boolean searchMatrix(int[][] matrix, int target) {
        int m = matrix.length;
        int n = matrix[0].length;
        int i=0;
        int j=n-1;
        while(i>=0 && j>=0 && i<m && j<n){
            if(target>matrix[i][j]) i++;
            else if(target<matrix[i][j]) j--;
            else return true;
        }
        return false;        
    }
}
```
