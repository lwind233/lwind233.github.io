## 深度优先搜索

搜索左右子树，记录其中是否出现过pq节点，然后判断自己是不是第一个发现左右子树都存在pq节点的节点，是的话就记录自己。

```java
class Solution {
    TreeNode fresult=null;
    public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
        traversal(root,p,q);
        return fresult;
    }

    public boolean[] traversal(TreeNode root, TreeNode p, TreeNode q){
        if(root==null) return new boolean[]{false,false};
        boolean[] right;
        boolean[] left;
        boolean[] result = new boolean[]{false,false};
        right = traversal(root.right,p,q);
        left = traversal(root.left,p,q);
        result[0] = right[0] || left[0] || root==p;
        result[1] = right[1] || left[1] || root==q;
        if(result[0]&&result[1]&&fresult==null) fresult=root;
        return result;

    }
}
```

## 路径记录法

遍历两次二叉树，分别记录从根节点到p和从根节点到q的路径，由于是递归遍历所以是倒序记录的。因此只需倒序遍历路径，并判断当前路径的节点是否在另一个路径中出现过，出现的第一个节点即为最近公共祖先

```java
class Solution {
    public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
        List<TreeNode> parentsP = new ArrayList<>();
        List<TreeNode> parentsQ = new ArrayList<>();

        traversal(parentsP,root,p);
        traversal(parentsQ,root,q);

        for(TreeNode t : parentsP){
            if(parentsQ.contains(t)) return t;
        }
        
        return null;
    }

    public boolean traversal(List<TreeNode> ls, TreeNode root, TreeNode target){
        if(root==null) return false;

        if(root.val==target.val){
            ls.add(root);
            return true;
        }

        if(traversal(ls,root.left,target)){
            ls.add(root);
            return true;
        }

        if(traversal(ls,root.right,target)){
            ls.add(root);
            return true;
        }

        return false;
    }
}
```
