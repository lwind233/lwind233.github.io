就是两数之和，不过复杂度高了一点，把前两个数组的和当键，查target-n3-n4是否存在，存在多少个。没啥需要讲的，看代码就完事了



```java
class Solution {
    public int fourSumCount(int[] nums1, int[] nums2, int[] nums3, int[] nums4) {
        HashMap<Integer,Integer> map12 = new HashMap<>();
        HashMap<Integer,Integer> map34 = new HashMap<>();


        for(int i=0;i<=nums1.length-1;i++){
            for(int j=0;j<=nums2.length-1;j++){
                map12.put(nums1[i]+nums2[j], map12.getOrDefault(nums1[i]+nums2[j],0)+1);
            }
        }
        int result=0;


        for(int i=0;i<=nums3.length-1;i++){
            for(int j=0;j<=nums4.length-1;j++){
                if(map12.containsKey(0-nums3[i]-nums4[j])){
                    result+=map12.get(0-nums3[i]-nums4[j]);
                }
            }
        }


        return result;


    }
}
```
