## 解法1
子字符串肯定是原字符串的前缀
子字符串重复n遍肯定是原字符串

易得

```java
class Solution {
    public boolean repeatedSubstringPattern(String s) {
        int len = s.length();
        int tmpLen;
        int repeatNum;
        String tmpS;
        StringBuilder sb = new StringBuilder();
        for(int i=0;i<=len-1;i++){
            sb.append(s.charAt(i));
            tmpLen = sb.length();
            if(len%tmpLen==0){
                repeatNum = len/tmpLen;
                if(repeatNum==1) return false;
                tmpS = "";
                for(int j=1;j<=repeatNum;j++){
                    tmpS = tmpS+sb.toString();
                }
                if(tmpS.equals(s)) return true;
            }
        }
        return false;
    }
}
```

## 解法2
假设一个字符串s由子串ss拼成，那么s+s去掉首位字符后肯定还包含一个s

```java
class Solution {
    public boolean repeatedSubstringPattern(String s) {
        String tmp = (s+s).substring(1,s.length()*2-1);
        return tmp.contains(s);
    }
}
```
