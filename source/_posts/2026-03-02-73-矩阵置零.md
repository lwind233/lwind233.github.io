用第一行和第一列作为标记，假设[i,j]处有0，则将[i][0]和[0][j]标记为0，代表这一行和这一列都要置为0

为了处理第一行和第一列，首先扫一遍第一行和第一列，判断有没有0，有的话用一个标记进行记录，最后若有这个标记，则第一行或第一列全置为0

随后遍历矩阵剩余部分，按先前约定的规则对第一行和第一列进行标记

随后根据第一行和第一列的标记，对矩阵进行置零

最后对第一行和第一列进行置零操作

```java
class Solution {
    public void setZeroes(int[][] matrix) {
        int cl=0;
        int hl=0;
        int w = matrix[0].length;
        int h = matrix.length;
        for(int i=0;i<=w-1;i++){
            if(matrix[0][i]==0) cl=1;
        }
        for(int i=0;i<=h-1;i++){
            if(matrix[i][0]==0) hl=1;
        }
        for(int j=1;j<=h-1;j++){
            for(int i=1;i<=w-1;i++){
                if(matrix[j][i]==0){
                    matrix[j][0]=0;
                    matrix[0][i]=0;
                }
            }
        }

        for(int i=1;i<=w-1;i++){
            if(matrix[0][i]==0){
                for(int j=1;j<=h-1;j++){
                    matrix[j][i]=0;
                }
            }
        }

        for(int i=1;i<=h-1;i++){
            if(matrix[i][0]==0){
                for(int j=1;j<=w-1;j++){
                    matrix[i][j]=0;
                }
            }
        }

        if(cl==1) for(int i=0;i<=w-1;i++) matrix[0][i]=0;
        if(hl==1) for(int i=0;i<=h-1;i++) matrix[i][0]=0;

    }
}
```
