仍然是双指针，一个指针指向上一个节点对的尾节点，另一个指针指向当前节点对的头结点

注意好赋值顺序即可，先用tmp存一下尾节点的下一个节点的位置，然后更新上一个节点对尾节点的下一节点，然后顺着往下更新当前节点对更新后的尾节点，最后调整当前尾节点的next值。最后更新pre和cur

如果cur.next是空值，也就是到链表尾部了（链表节点数量为奇数），直接退出

画图的话解释的会更方便，但这里不方便画图，看代码吧

```java
class Solution {
    public ListNode swapPairs(ListNode head) {
        if(head==null || head.next==null) return head;
        ListNode tHead = new ListNode();
        tHead.next = head;
        ListNode pre = tHead;
        ListNode cur = tHead.next;
        ListNode tmp;
        while(cur!=null){
            if(cur.next==null) break;
            tmp = cur.next.next;
            pre.next = cur.next;
            cur.next.next = cur;
            cur.next = tmp;


            pre = cur;
            cur = cur.next;
        }


        return tHead.next;
    }
}
```
