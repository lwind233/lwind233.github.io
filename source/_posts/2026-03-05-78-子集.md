## 迭代法

每次在上一次的结果上把新的元素加到结果里的每一个列表里，和原结果列表合并就是加入该元素后的子集列表

```java
class Solution {
    LinkedList<Integer> path;
    List<List<Integer>> result;
    public List<List<Integer>> subsets(int[] nums) {
        path = new LinkedList<>();
        result = new ArrayList<>();
        result.add(new ArrayList<>());
        
        for(int i=0;i<=nums.length-1;i++){
            List<List<Integer>> tmpL = new ArrayList<>();
            for(List ls:result){
                List tmp = new ArrayList<>(ls);
                tmp.add(nums[i]);
                tmpL.add(tmp);
            }
            result.addAll(tmpL);
        }
        return result;
    }
}
```

## 回溯
每次选择一个元素后，接下来的回溯过程要从这个元素的下一位开始选择

```java
class Solution {
    LinkedList<Integer> path;
    List<List<Integer>> result;

    public List<List<Integer>> subsets(int[] nums) {
        path = new LinkedList<>();
        result = new ArrayList<>();
        backtrace(nums,0);
        return result;
    }

    public void backtrace(int[] nums, int idx){
        result.add(new ArrayList<>(path));

        for(int i=idx;i<=nums.length-1;i++){
            path.add(nums[i]);
            backtrace(nums,i+1);
            path.removeLast();
        }
    }
}
```

回溯还有一种做法是，依次遍历每个元素，每个元素选择不加入或加入，然后进行下一层的判断，如果不加入不需要回溯，但加入就要进行回溯以撤销影响

```java
class Solution {
    List<List<Integer>> res = new ArrayList<>();
    LinkedList<Integer> path = new LinkedList<>();

    public List<List<Integer>> subsets(int[] nums) {
        dfs(nums, 0);
        return res;
    }

    private void dfs(int[] nums, int idx) {
        if (idx == nums.length) {
            res.add(new ArrayList<>(path));
            return;
        }

        // 不选 nums[idx]
        dfs(nums, idx + 1);

        // 选 nums[idx]
        path.add(nums[idx]);
        dfs(nums, idx + 1);
        path.removeLast();
    }
}
```
