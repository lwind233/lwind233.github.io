重点是搞清旋转前后的坐标关系，也就是
```java
tmp[0] = matrix[t1][t2];
tmp[1] = matrix[t2][n-1-t1];
tmp[2] = matrix[n-1-t1][n-1-t2];
tmp[3] = matrix[n-1-t2][t1];
```
剩余的内容就是确定每一层的起点，以及确定从外到内每一层要进行几次旋转，最外层是旋转从开始到倒数第二位元素，每往内一层旋转的元素数目少两个

```java
class Solution {
    public void rotate(int[][] matrix) {
        int n = matrix[0].length;
        int s1=0;
        int s2=0;
        int[] tmp = new int[4];

        for(int i=1;i<=(n+1)/2;i++){
            for(int j=0;j+s2<=n-1-i;j++){
                int t1 = s1;
                int t2 = s2+j;

                tmp[0] = matrix[t1][t2];
                tmp[1] = matrix[t2][n-1-t1];
                tmp[2] = matrix[n-1-t1][n-1-t2];
                tmp[3] = matrix[n-1-t2][t1];

                matrix[t1][t2] = tmp[3];
                matrix[t2][n-1-t1] = tmp[0];
                matrix[n-1-t1][n-1-t2] = tmp[1];
                matrix[n-1-t2][t1] = tmp[2];
            }
            s1++;
            s2++;
        }
    }
}
```
