遇到左括号就入栈
遇到右括号就peek一下栈，栈空直接返回fasle。看看栈顶是不是和当前括号匹配，匹配出栈，不匹配返回false
最后检查当前栈是否为空
```java
class Solution {
    public boolean isValid(String s) {
        Stack<Character> k = new Stack<>();
        int len = s.length();
        for(int i=0;i<=len-1;i++){
            if(s.charAt(i)=='(' || s.charAt(i)=='{' || s.charAt(i)=='['){
                k.push(s.charAt(i));
            }else{
                if(k.isEmpty()) return false;
                char tmp = k.peek();
                if(s.charAt(i)==')' && tmp=='(') k.pop();
                else if(s.charAt(i)==']' && tmp=='[') k.pop();
                else if(s.charAt(i)=='}' && tmp=='{') k.pop();
                else return false;
            }
        }
        return k.isEmpty();
    }
}
```
