维护一个滑动窗口，窗口从左向右遍历，每当右边界向右拓展一个新元素时，左边界也尽量向右收缩，同时满足窗口内元素大于目标值的要求

```java
class Solution {
    public int minSubArrayLen(int target, int[] nums) {
        int left=0;
        int len = nums.length;
        int sum=0;
        int minLen=len;
        for(int right=0;right<=len-1;right++){
            sum = sum+nums[right];
            if(sum>=target){
                while(true){
                    int tmp = nums[left];
                    if(sum-tmp>=target){
                        sum-=tmp;
                        left++;
                    }else{
                        break;
                    }
                }
                int tmpLen = right-left+1;
                if(tmpLen<minLen){
                    minLen = tmpLen;
                }
            }
        }
        if(sum<target) return 0;

        return minLen;
    }
}
```
