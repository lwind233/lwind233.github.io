递归解决，以空节点高度为0，每层的高度等于左右子节点的最大高度加一，每层的最大路径长等于左右子树中的最大路径与左右高度之和的最大值。

不过确实有点绕，二刷的时候把高度的定义理解混了，写出了中等题的感觉

```java
class Solution {
    public int diameterOfBinaryTree(TreeNode root) {
        return traversal(root)[1];
    }

    public int[] traversal(TreeNode root){
        if(root==null) return new int[]{0,0};
        int[] result = new int[2];
        int[] left = traversal(root.left);
        int[] right = traversal(root.right);

        result[0] = Math.max(left[0],right[0])+1;
        result[1] =Math.max(Math.max(left[1],right[1]),left[0]+right[0]);
        return result;
    }
}
```
