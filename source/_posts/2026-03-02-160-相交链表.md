## 对齐法

先找出长短链表，然后对齐两个链表的起点，随后同时开始遍历链表，找出两个链表在遍历中是否会遇见相同的结点（交点）

```java
public class Solution {
    public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
        int lenA=0;
        int lenB=0;
        ListNode cur = headA;
        while(cur!=null){
            lenA++;
            cur = cur.next;
        }

        cur=headB;
        while(cur!=null){
            lenB++;
            cur = cur.next;
        }

        ListNode longList;
        ListNode shortList;
        int diff = 0;
        if(lenA>=lenB){
            longList = headA;
            shortList = headB;
            diff = lenA-lenB;
        }else{
            longList = headB;
            shortList = headA;
            diff = lenB-lenA;
        }

        ListNode cur1 = shortList;
        ListNode cur2 = longList;
        for(int i=1;i<=diff;i++) cur2 = cur2.next;
        while(cur1!=null){
            if(cur1==cur2) return cur1;
            cur1 = cur1.next;
            cur2 = cur2.next;
        }
        return null;

    }
}
```

## 双指针法
核心思路

用两个指针 pA、pB 分别从 headA、headB 出发：

* 每次走一步

* 走到末尾（null）就跳到另一条链表的头继续走

这样两个指针走过的总路程都是 lenA + lenB：

* 如果有交点，会在交点相遇

* 如果没交点，会同时走到 null 相遇（返回 null）

时间 O(lenA + lenB)，空间 O(1)。


```java
public class Solution {
    public ListNode getIntersectionNode(ListNode headA, ListNode headB) {
        if (headA == null || headB == null) return null;

        ListNode pA = headA, pB = headB;
        while (pA != pB) {
            pA = (pA == null) ? headB : pA.next;
            pB = (pB == null) ? headA : pB.next;
        }
        return pA; // 相交点或 null
    }
}
```
