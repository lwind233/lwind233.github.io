维护快慢指针遍历链表，快指针追上慢指针说明有环，快指针碰到null说明没有环

```java
public class Solution {
    public boolean hasCycle(ListNode head) {
        ListNode slow=head;
        ListNode fast=head;
        while(fast!=null && slow!=null){
            slow = slow.next;
            fast = fast.next;
            if(fast==null) return false;
            fast = fast.next;
            if(fast==slow) return true;
        }
        return false;
    }
}
```
